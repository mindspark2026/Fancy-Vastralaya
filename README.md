# Fancy Vastralaya Website

This is the official website project for **Fancy Vastralaya**, a local clothing store located in **Chawal Bazar, Daudnagar, Bihar (India)**.
The website provides information about the shop, products available, and contact details so customers can easily connect with the business.

## 🏪 About the Shop

**Fancy Vastralaya** is a trusted clothing and fabric store that offers a variety of textile products at affordable prices. The shop serves customers in Daudnagar and nearby areas.
our website :-https://mindspark2026.github.io/Fancy-Vastralaya/
